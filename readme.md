### a k transpiler
d uses shakti's k.edu core for primitives

### get, build and run
```
$ git clone --recursive https://github.com/effbiae/d.git   #recursive to get the k.edu submodule
$ cd d
$ make
$ ./d
```

### limitations
```
implemented adverbs:
  iterate  nf\x     3%:\3
  each     f'x      !:'!3
  scan     F\x      +\!3
  over     F/x      +/!3
functions are monadic (only mention x)
```
